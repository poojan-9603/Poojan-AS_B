In this experiment, you will solve mini Sudoku puzzles.

The rules of Sudoku are to fill in all the digits so that no duplicates appear in any row, column or bold-bordered square.

This version supports both mouse and keyboard input:

  • Mouse: Click a number on the side, then click a square
  • Keyboard: Use arrow keys to navigate and number keys (1-9) to input values
  • Press Tab to jump to next empty cell or use arrow keys to navigate
  • Press Backspace/Delete to clear a cell
  • If a input turns or blinks red then you need to recheck it
  • However if the input turns or blinks the cell green then you are good for now
  • but that doesn't mean you can't screw it later on.
